# 成就系统测试指南

## 功能验证清单

### 1. 难度系统测试
- [ ] 创建高难度任务，验证是否显示红色标签
- [ ] 创建中等难度任务，验证是否显示橙色标签  
- [ ] 创建低难度任务，验证是否显示绿色标签
- [ ] 完成不同难度的任务，验证状态更新

### 2. 成就创建测试
- [ ] 点击头像进入个人资料页面
- [ ] 切换到"成就管理"标签
- [ ] 点击添加成就按钮
- [ ] 测试每种成就条件类型：

#### 完成任务数量
1. 选择"完成任务数量"
2. 输入目标数量（如10）
3. 验证创建成功

#### 完成指定难度任务 ⭐ (新功能)
1. 选择"完成指定难度任务"
2. **验证难度选择下拉菜单出现**
3. 选择难度：低/中/高
4. 输入目标数量
5. 验证创建成功且包含难度信息

#### 连续完成任务
1. 选择"连续完成任务"
2. 输入连续天数
3. 验证创建成功

#### 专精特定职业
1. 选择"专精特定职业"
2. 从职业列表中选择
3. 验证创建成功

### 3. 成就进度测试
- [ ] 创建测试成就后，执行相应操作
- [ ] 验证成就进度实时更新
- [ ] 验证完成成就时的状态变化

### 4. 数据库一致性测试
- [ ] 重启应用验证数据持久化
- [ ] 验证难度信息正确保存和加载
- [ ] 验证成就与职业关联正确

## 测试用例示例

### 用例1：创建"完成10个高难度任务"成就
1. 进入成就管理页面
2. 点击添加成就
3. 填写基本信息：
   - 名称：高难度专家
   - 描述：完成10个高难度任务
   - 图标：选择一个图标
   - 颜色：选择红色
4. 选择条件类型："完成指定难度任务"
5. **关键步骤：选择难度为"高"**
6. 输入目标数量：10
7. 保存成就
8. 验证成就列表中显示新成就

### 用例2：验证难度条件生效
1. 创建几个高难度任务
2. 完成这些任务
3. 检查"高难度专家"成就的进度
4. 验证只有高难度任务计入进度

## 预期结果
- 所有难度级别正确显示对应颜色
- 成就创建表单根据条件类型显示相应字段
- 难度选择功能正常工作
- 成就进度正确跟踪指定难度的任务
- 数据持久化正常
